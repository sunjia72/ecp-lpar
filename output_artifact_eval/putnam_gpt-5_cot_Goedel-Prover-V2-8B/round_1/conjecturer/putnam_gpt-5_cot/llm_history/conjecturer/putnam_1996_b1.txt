Let \(s\subseteq \{1,\dots,n\}\) be “selfish”, so by definition
\[
\text{selfish}(s)\iff |s|\in s.
\]
We want the selfish sets that are minimal under proper inclusion.

If \(|s|=m\), then selfishness means \(m\in s\).  
Now suppose some \(k\in s\) also satisfies \(k<m\). Then we can choose a proper subset of \(s\) of size \(k\) that still contains \(k\), so that subset is selfish too. This contradicts minimality.

So in a minimal selfish set of size \(m\), the only element \(\le m\) is \(m\) itself. Hence such a set must have the form
\[
s=\{m\}\cup T,\qquad T\subseteq \{m+1,\dots,n\},\quad |T|=m-1.
\]
Conversely, every such set is clearly minimal selfish, since every proper subset has size \(<m\) but contains no element equal to its size.

Therefore, for each \(m\), the number of minimal selfish sets of size \(m\) is
\[
\binom{n-m}{m-1}.
\]
Summing over all possible \(m\),
\[
\sum_{m\ge 1}\binom{n-m}{m-1}
=\sum_{r\ge 0}\binom{n-r-1}{r},
\]
which is the standard Fibonacci identity giving \(F_n\) (with \(F_1=1,F_2=1\)). In Lean this is `Nat.fib n`.
