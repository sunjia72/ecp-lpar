Let  
\[
A_n(x)=\frac1n\sum_{i<n}x_i
\]
be the Cesàro mean.

The condition says that for every sequence \(x_i\in[0,1]\), if \(A_n(x)\) converges, then \(A_n(f\circ x)\) also converges.

A linear function on \([0,1]\), say \(f(x)=ax+b\), clearly works, since
\[
A_n(f(x_i))=aA_n(x_i)+b.
\]

So the main issue is necessity. If \(f\) is not affine on \([0,1]\), then there exist \(u,v\in[0,1]\) and \(t\in[0,1]\) such that
\[
f(tu+(1-t)v)\neq tf(u)+(1-t)f(v).
\]
Using a sequence with asymptotic density \(t\) of \(u\)'s and \(1-t\) of \(v\)'s, and alternating very long blocks of that sequence with very long constant blocks at the value \(tu+(1-t)v\), one can make the original Cesàro mean still converge, while the Cesàro mean of \(f(x_i)\) oscillates between the two unequal quantities above, hence does not converge. Contradiction.

Therefore \(f\) must satisfy
\[
f(tx+(1-t)y)=tf(x)+(1-t)f(y)\qquad (x,y,t\in[0,1]).
\]
Taking \(x=1,\ y=0,\ t=z\) gives
\[
f(z)=z f(1)+(1-z)f(0)\qquad (z\in[0,1]),
\]
so \(f\) is affine on \([0,1]\). Outside \([0,1]\), the condition imposes nothing.

Thus the solution set is exactly the set of functions whose restriction to \([0,1]\) is affine.
