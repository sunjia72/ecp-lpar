\[
a_1 < a_2 < \cdots < a_m.
\]
If \(m\ge 2\), then two consecutive elements cannot have the same parity, because then their midpoint would be an integer strictly between them and would have to lie in \(S\), contradiction. So consecutive differences are all odd.

If \(m\ge 3\), then \(a_i\) and \(a_{i+2}\) have the same parity, so their midpoint must lie in \(S\). The only element of \(S\) between them is \(a_{i+1}\), hence
\[
a_{i+1}=\frac{a_i+a_{i+2}}2,
\]
so the consecutive differences are equal. Therefore every mediocre set with at least 2 elements is an arithmetic progression with odd common difference; conversely every such progression is mediocre.

- singletons,
