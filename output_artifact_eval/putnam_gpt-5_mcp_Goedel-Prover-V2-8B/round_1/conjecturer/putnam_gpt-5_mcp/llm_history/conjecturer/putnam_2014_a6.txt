Reason:

- For each \(i\), define a polynomial in the entries \(x_{qp}\) of an \(n\times n\) matrix \(X\):
  \[
  P_i(X)=\prod_{p=1}^n \left(\sum_{q=1}^n (M_i)_{pq}x_{qp}\right).
  \]
  Then
  \[
  P_i(N_j)=\prod_{p=1}^n (M_iN_j)_{pp}.
  \]
  By the hypothesis, this is nonzero when \(i=j\), and zero when \(i\ne j\). So the \(P_i\) are linearly independent.

- But each \(P_i\) is multilinear, choosing one variable from each column-block \(p\). So all such polynomials lie in a vector space with basis
  \[
  x_{a_1,1}x_{a_2,2}\cdots x_{a_n,n},
  \]
  where each \(a_p\in\{1,\dots,n\}\). Hence the dimension is \(n^n\). Therefore \(k\le n^n\).

- For equality, index matrices by functions \(a:\{1,\dots,n\}\to\{1,\dots,n\}\) (there are \(n^n\) of these). Let \(M_a\) have row \(p\) equal to \(e_{a(p)}^T\), and \(N_a\) have column \(p\) equal to \(e_{a(p)}\). Then
  \[
  (M_aN_b)_{pp}=e_{a(p)}^Te_{b(p)}=
  \begin{cases}
  1,& a(p)=b(p),\\
  0,& a(p)\ne b(p).
  \end{cases}
  \]
  So if \(a=b\), all diagonal entries are \(1\); if \(a\ne b\), some diagonal entry is \(0\). Thus \(k=n^n\) is achievable.
