B=A-A^T.

For a uniformly random \(0\)-\(1\) matrix \(A\) of size \(2n\times 2n\), the average in the statement is exactly \(\mathbb E[\det(B)]\).

\[

\]
Then the \(X_{ij}\) are independent, with
\[
\Pr(X_{ij}=1)=\Pr(X_{ij}=-1)=\tfrac14,\qquad \Pr(X_{ij}=0)=\tfrac12,
\]
so
\[
\mathbb E[X_{ij}]=0,\qquad \mathbb E[X_{ij}^2]=\tfrac12.
\]
Also \(B_{ji}=-X_{ij}\) and \(B_{ii}=0\).

Now expand the determinant:
\[
\det(B)=\sum_{\sigma\in S_{2n}}\operatorname{sgn}(\sigma)\prod_{i=1}^{2n} B_{i,\sigma(i)}.
\]
Taking expectations termwise:

- If \(\sigma\) has a fixed point, then the term is \(0\) because \(B_{ii}=0\).
- If \(\sigma\) has a cycle of length \(\ge 3\), then some \(X_{ij}\) appears only once, so the expectation is \(0\) since \(\mathbb E[X_{ij}]=0\).
- Thus only permutations that are products of \(n\) disjoint transpositions contribute, i.e. perfect matchings.

For one transposition \((ij)\),
\[
\operatorname{sgn}((ij))\,\mathbb E[B_{ij}B_{ji}]
= (-1)\,\mathbb E[-X_{ij}^2]
= \mathbb E[X_{ij}^2]
= \tfrac12.
\]
So each perfect matching contributes \((1/2)^n\).

The number of perfect matchings on \(2n\) labeled points is
\[
\frac{(2n)!}{2^n n!}.
\]
Hence
\[
\mathbb E[\det(B)]
=
\frac{(2n)!}{2^n n!}\left(\frac12\right)^n
=
\frac{(2n)!}{4^n n!}.
\]
