Let \(f(n)\) be the number of \(n\times n\) matrices \(A=(a_{ij})\) with entries in \(\{-1,0,1\}\) such that
\[
\sum_i a_{i,\phi(i)}
\]
is the same for every permutation \(\phi\in S_n\).

Using Python, I checked the first values:

- \(f(1)=3\)
- \(f(2)=19\)
- \(f(3)=87\)
- \(f(4)=355\)

and these match
\[
4^n+2\cdot 3^n-4\cdot 2^n+1.
\]

## Reasoning

For \(n\ge 2\), compare two permutations differing only by swapping the columns chosen in two rows. This gives, for all rows \(i,k\) and columns \(j,\ell\),
\[
a_{ij}+a_{k\ell}=a_{i\ell}+a_{kj}.
\]
So every \(2\times 2\) submatrix has equal diagonal sums.

Hence \(A\) has the form
\[
a_{ij}=r_i+c_j
\]
for some integers \(r_i,c_j\). Conversely, any such matrix has constant permutation sum, since
\[
\sum_i (r_i+c_{\phi(i)})=\sum_i r_i+\sum_j c_j.
\]

So we just count matrices of the form \(r_i+c_j\) with all values in \(\{-1,0,1\}\).

Normalize by subtracting a constant so that \(\min_j c_j=0\). Then necessarily \(c_j\in\{0,1,2\}\).

### Case 1: \(\max c_j=0\)
Then all \(c_j=0\), so each row is constant:
\[
3^n
\]
choices.

### Case 2: \(\max c_j=1\)
Then \(c_j\in\{0,1\}\), with both 0 and 1 appearing. To keep \(r_i+c_j\in\{-1,0,1\}\), we need \(r_i\in\{-1,0\}\).

- rows: \(2^n\) choices
- columns: any nonconstant \(\{0,1\}\)-sequence: \(2^n-2\)

So:
\[
2^n(2^n-2).
\]

### Case 3: \(\max c_j=2\)
Then \(c_j\in\{0,1,2\}\), with both 0 and 2 appearing. This forces \(r_i=-1\) for every \(i\).

Number of column choices:
\[
3^n-2\cdot 2^n+1
\]
(by inclusion-exclusion: all \(\{0,1,2\}\)-sequences containing both 0 and 2).

So total:
\[
f(n)=3^n+2^n(2^n-2)+(3^n-2\cdot 2^n+1)
=4^n+2\cdot 3^n-4\cdot 2^n+1.
\]
