Let  
\[
F(a,x)=\sum_{n=0}^\infty (n+1)^a e^{2(n+1)}x^{(n+1)^2},\qquad 0\le x<1.
\]

Set \(m=n+1\) and \(t=-\log x\), so \(t\to 0^+\) as \(x\to 1^{-}\). Then
\[
F(a,x)=\sum_{m=1}^\infty m^a e^{2m-tm^2}.
\]
Complete the square:
\[
2m-tm^2=\frac1t-t\Bigl(m-\frac1t\Bigr)^2.
\]
Hence
\[
F(a,x)=e^{1/t}\sum_{m=1}^\infty m^a e^{-t(m-1/t)^2}.
\]

The Gaussian is centered at \(m\approx 1/t\) with width \(\approx 1/\sqrt t\), so on the main range \(m\sim 1/t\), we have \(m^a\sim t^{-a}\). Therefore, by the usual Gaussian/Riemann-sum heuristic,
\[
\sum_{m=1}^\infty m^a e^{-t(m-1/t)^2}
\sim t^{-a}\cdot \frac1{\sqrt t}\int_{-\infty}^{\infty} e^{-y^2}\,dy
= \sqrt{\pi}\, t^{-a-\frac12}.
\]
So
\[
F(a,x)\sim \sqrt{\pi}\, e^{1/t} t^{-a-\frac12}.
\]

Now compare \(t=-\log x\) with \(1-x\). Since
\[
-\log x = (1-x)+\frac{(1-x)^2}{2}+O((1-x)^3),
\]
we get
\[
\frac1t-\frac1{1-x}\to -\frac12.
\]
Thus
\[
e^{1/t}e^{-1/(1-x)}\to e^{-1/2}.
\]
Therefore
\[
F(a,x)e^{-1/(1-x)}
\sim \sqrt{\frac{\pi}{e}}\, t^{-a-\frac12}
\sim \sqrt{\frac{\pi}{e}}\,(1-x)^{-a-\frac12}.
\]

So the behavior is determined by the exponent \(-a-\tfrac12\):
- if \(a<-\tfrac12\), then \((1-x)^{-a-1/2}\to 0\);
- if \(a>-\tfrac12\), then \((1-x)^{-a-1/2}\to \infty\).

Hence the unique threshold is
\[
c=-\frac12.
\]
