Let  
\[
A=\prod_{i\in n} i,\qquad F(x)=\frac{1}{A}\prod_{i\in n}(x+i).
\]
The hypothesis says exactly that \(F(k)\in \mathbb Z\) for every integer \(k\).

Python check on many small examples supports:
- every such set contains \(1\) or \(-1\),
- every positive such set is \(\{1,2,\dots,|n|\}\).

Mathematical reason:

1. **Must contain \(1\) or \(-1\).**  
   If neither \(1\) nor \(-1\) is in \(n\), then every \(i\in n\) has \(|i|\ge 2\). Hence
   \[
   F(1)F(-1)=\prod_{i\in n}\left(1-\frac1{i^2}\right).
   \]
   Each factor is strictly between \(0\) and \(1\), so the whole product is strictly between \(0\) and \(1\).  
   But \(F(1),F(-1)\in\mathbb Z\), so \(F(1)F(-1)\in\mathbb Z\), impossible.  
   Therefore \(1\in n\) or \(-1\in n\).

2. **If all elements are positive, then \(n=\{1,2,\dots,|n|\}\).**  
   Write the positive elements as
   \[
   1\le a_1<a_2<\cdots<a_m,\qquad m=n.card.
   \]
   Then \(A=a_1a_2\cdots a_m\), and \(F(x)\) is an integer-valued polynomial of degree \(m\) with leading coefficient \(1/A\).  
   A standard fact about integer-valued polynomials says: for degree \(m\), \(m!\) times the leading coefficient is an integer. So
   \[
   \frac{m!}{A}\in\mathbb Z,
   \]
   i.e. \(A\mid m!\).

   But since \(a_j\ge j\) for each \(j\),
   \[
   A=a_1\cdots a_m \ge 1\cdot 2\cdot \cdots \cdot m = m!.
   \]
   Since \(A\) divides \(m!\) and \(A\ge m!\), we must have \(A=m!\). Thus every inequality \(a_j\ge j\) is equality, so \(a_j=j\) for all \(j\). Therefore
   \[
   n=\{1,2,\dots,m\}=\mathrm{Finset.Icc}\ 1\ m.
   \]

So both statements are **true**.
