Using Perron–Frobenius:

Let \(A=(a_{ij})\). Since all \(a_{ij}>0\), \(A^T\) has a positive eigenvector \(b\neq 0\) with positive eigenvalue \(\lambda>0\):
\[
A^T b = \lambda b.
\]

Define
\[
y(t)=\sum_i b_i x_i(t).
\]
Then
\[
y'(t)=\sum_i b_i x_i'(t)
=\sum_i b_i\sum_j a_{ij}x_j(t)
=\sum_j (A^Tb)_j x_j(t)
=\lambda \sum_j b_j x_j(t)
=\lambda y(t).
\]
So \(y\) satisfies \(y'=\lambda y\), hence \(y(t)=Ce^{\lambda t}\).

But each \(x_i(t)\to 0\) as \(t\to+\infty\), so also \(y(t)\to 0\). Since \(\lambda>0\), the only solution \(Ce^{\lambda t}\) tending to \(0\) at \(+\infty\) is \(C=0\). Thus \(y(t)\equiv 0\), i.e.
\[
\sum_i b_i x_i(t)=0 \quad \text{for all } t,
\]
with \(b\neq 0\). Therefore the functions \(x_i\) are always linearly dependent.

So the proposition in the statement is **true**.
