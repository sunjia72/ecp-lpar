Let  
\[
f(x)=p(x)-x.
\]
Then the condition is
\[
p(p(x))-x=(p(x)-x)^2q(x),
\]
i.e.
\[
f(x)+f(p(x))=f(x)^2q(x).
\]
So \(f(x)^2\) divides \(f(x)+f(p(x))\).

Work over \(\mathbb C\) (same polynomial identity).

Take any root \(\alpha\) of \(f\), and let its multiplicity be \(m\).

- If \(m\ge 2\), write
  \[
  f(x)=c(x-\alpha)^m+\cdots,\qquad c\neq 0.
  \]
  Since \(p(x)=x+f(x)\),
  \[
  p(x)-\alpha=(x-\alpha)+c(x-\alpha)^m+\cdots,
  \]
  so
  \[
  f(p(x))=c(x-\alpha)^m+\cdots.
  \]
  Hence
  \[
  f(x)+f(p(x))=2c(x-\alpha)^m+\cdots,
  \]
  which has order \(m\), not at least \(2m\), contradiction.  
  So every root of \(f\) is simple.

- Thus for a root \(\alpha\),
  \[
  f(x)=c(x-\alpha)+\cdots,\qquad c=f'(\alpha)\neq 0.
  \]
  Then
  \[
  p(x)-\alpha=(1+c)(x-\alpha)+\cdots,
  \]
  so
  \[
  f(p(x))=c(1+c)(x-\alpha)+\cdots.
  \]
  Therefore
  \[
  f(x)+f(p(x))=c(c+2)(x-\alpha)+\cdots.
  \]
  Since this must be divisible by \(f(x)^2\), the linear term must vanish:
  \[
  c(c+2)=0.
  \]
  As \(c\neq 0\), we get \(c=-2\). So every root \(\alpha\) of \(f\) satisfies \(f'(\alpha)=-2\).

If \(\deg f=n\ge 2\), then \(f\) has \(n\) distinct complex roots, and each is a root of \(f'+2\), but \(\deg(f'+2)=n-1\), impossible. Hence \(\deg f\le 1\), so \(p\) is affine:
\[
p(x)=ax+b.
\]

Now check affine polynomials:

- If \(a=1\), then \(p(x)=x+b\), and
  \[
  p(p(x))-x=2b.
  \]
  Since \(p(x)-x=b\), this works (take \(q=0\) if \(b=0\), and \(q=2/b\) if \(b\neq 0\)).

- If \(a\neq 1\), then \(p(x)-x=(a-1)x+b\) is linear, and
  \[
  p(p(x))-x=(a+1)(p(x)-x).
  \]
  For this to be divisible by \((p(x)-x)^2\), we must have \(a+1=0\), so \(a=-1\). Thus \(p(x)=b-x\), and then \(p(p(x))=x\), so it works.

So the solution set is exactly all
\[
p(x)=x+b \quad\text{or}\quad p(x)=b-x \qquad (b\in\mathbb R).
\]
