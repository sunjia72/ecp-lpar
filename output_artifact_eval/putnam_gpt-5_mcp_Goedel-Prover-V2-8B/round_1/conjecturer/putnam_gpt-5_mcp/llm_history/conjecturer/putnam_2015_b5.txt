Let  
\[
P(n)
\]
be the number of permutations \(\pi\) of \(\{1,\dots,n\}\) such that adjacent integers in the domain go to numbers at distance at most \(2\), i.e.
\[
|\pi(i+1)-\pi(i)|\le 2\quad (1\le i<n).
\]
So \(P(n)\) counts Hamiltonian paths in the graph on \(\{1,\dots,n\}\) with edges between numbers differing by \(1\) or \(2\).

Define \(a_n\) = number of such permutations that start with \(1\).

From the structure of such paths:

- If a path starts with \(1,2\), deleting the initial \(1\) gives a path of size \(n-1\) starting with \(1\): \(a_{n-1}\) choices.
- If it starts \(1,3,2,4\), deleting \(1,2,3\) gives a path of size \(n-3\) starting with \(1\): \(a_{n-3}\) choices.
- The only remaining possibility is the unique zigzag
  \[
  1,3,5,\dots,n,n-1,n-3,\dots,2,
  \]
  giving \(1\) extra path.

Hence for \(n\ge 4\),
\[
a_n=a_{n-1}+a_{n-3}+1.
\]

Now count by first value. If a valid permutation starts with \(k\) (\(1<k<n\)), then it must first go to \(k-2\) or \(k+2\); these contribute \(a_{n-k}\) and \(a_{k-1}\) respectively. Therefore
\[
P(n)=2a_n+2\sum_{j=1}^{n-2} a_j .
\]

\[
\begin{aligned}
\frac{1}{2}\bigl(P(n+5)-P(n+4)-P(n+3)+P(n)\bigr)
&= a_{n+5}-a_{n+4}-a_{n+1}-a_{n-1}.
\end{aligned}

\[
a_{n+5}-a_{n+4}=a_{n+2}+1=(a_{n+1}+a_{n-1}+1)+1.

So
\[
a_{n+5}-a_{n+4}-a_{n+1}-a_{n-1}=2.
