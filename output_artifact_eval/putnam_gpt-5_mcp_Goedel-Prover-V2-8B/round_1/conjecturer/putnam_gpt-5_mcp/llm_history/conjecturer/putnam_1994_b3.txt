Let  
\[
g(x)=f(x)e^{-x}.
\]
Then
\[
g'(x)=e^{-x}(f'(x)-f(x)).
\]
So the hypothesis \(0<f(x)<f'(x)\) for all \(x\) is exactly the statement that \(g(x)>0\) and \(g'(x)>0\) for all \(x\). Hence every such \(f\) has the form
\[
f(x)=e^x g(x)
\]
where \(g\) is positive and strictly increasing.

Now:

- If \(k<1\), then \(e^{(k-1)x}\to 0\) as \(x\to\infty\). Since \(g\) is positive increasing, for \(x\ge 0\) we have \(g(x)\ge g(0)>0\). So for large enough \(x\),
  \[
  e^{(k-1)x}<g(0)\le g(x),
  \]
  hence
  \[
  e^{kx}=e^x e^{(k-1)x}<e^x g(x)=f(x).
  \]
  Thus every such \(f\) eventually dominates \(e^{kx}\).

- If \(k\ge 1\), take
  \[
  g(x)=\frac{1}{1+e^{-x}},
  \]
  which is positive, differentiable, strictly increasing, and satisfies \(0<g(x)<1\). Then
  \[
  f(x)=e^x g(x)
  \]
  satisfies \(0<f(x)<f'(x)\), but also \(f(x)<e^x\) for all \(x\). So:
  - for \(k=1\), \(e^x<f(x)\) is never true;
  - for \(k>1\), \(e^{kx}>e^x>f(x)\) for all sufficiently large \(x\).
  
So the set of all such \(k\) is exactly
\[
\{k\in\mathbb R: k<1\}.
\]
