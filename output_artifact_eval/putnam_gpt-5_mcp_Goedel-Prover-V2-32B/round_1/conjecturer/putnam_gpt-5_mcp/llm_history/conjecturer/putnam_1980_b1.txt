Let
\[
\cosh x=\frac{e^x+e^{-x}}2.
\]
We want all real \(c\) such that
\[
\forall x\in \mathbb R,\qquad \cosh x \le e^{c x^2}.
\]

Using Python to inspect
\[
\frac{\log(\cosh x)}{x^2},
\]
the values decrease from near \(0.5\) as \(x\to 0\):

- \(x=0.001\): \(0.4999999168\)
- \(x=0.01\): \(0.4999916669\)
- \(x=0.1\): \(0.4991688822\)
- \(x=1\): \(0.4337808305\)

So the threshold is very plausibly \(c=\frac12\).

### Why \(c \ge \frac12\) works
It is enough to prove
\[
\cosh x \le e^{x^2/2}\qquad (\forall x).
\]
Since both sides are even, assume \(x\ge 0\). Define
\[
g(x)=e^{-x^2/2}\cosh x.
\]
Then
\[
g'(x)=e^{-x^2/2}(\sinh x-x\cosh x)
     =e^{-x^2/2}\cosh x(\tanh x-x).
\]
Now \(h(x)=x-\tanh x\) satisfies
\[
h'(x)=1-\operatorname{sech}^2 x=\tanh^2 x\ge 0,\qquad h(0)=0,
\]
so \(h(x)\ge 0\), i.e. \(\tanh x\le x\). Hence \(g'(x)\le 0\), so \(g(x)\le g(0)=1\). Therefore
\[
\cosh x \le e^{x^2/2}.
\]
If \(c\ge \frac12\), then \(e^{x^2/2}\le e^{c x^2}\), so the inequality holds for all \(x\).

### Why \(c < \frac12\) fails
Let
\[
F(x)=e^{c x^2}-\cosh x.
\]
Then
\[
F(0)=0,\qquad F'(0)=0,\qquad F''(0)=2c-1<0.
\]
So for sufficiently small nonzero \(x\), \(F(x)<0\). Hence
\[
\cosh x > e^{c x^2}
\]
for small \(x\), and the inequality fails.

Thus the solution set is exactly
\[
\{c\in\mathbb R : c\ge \tfrac12\}=[\tfrac12,\infty).
\]
