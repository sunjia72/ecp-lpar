Let  
\[
F_{a,b}(x)=\log(1+x^2)-ax-b.
\]
We want the pairs \((a,b)\) for which \(F_{a,b}(x)=0\) has exactly one real solution.

Python checks on many random samples matched the classification below.

### Key facts
- \(F'_{a,b}(x)=\dfrac{2x}{1+x^2}-a\).
- Since \(\log(1+x^2)=o(|x|)\), for \(a>0\),
  \[
  F_{a,b}(x)\to +\infty \text{ as }x\to-\infty,\qquad F_{a,b}(x)\to -\infty \text{ as }x\to+\infty,
  \]
  and similarly with signs reversed for \(a<0\).
- The equation is symmetric under \(x\mapsto -x\), so the answer depends only on \(|a|\).

### Cases
1. **\(|a|>1\)**:  
   \(\frac{2x}{1+x^2}\in[-1,1]\), so \(F'\) never vanishes and has constant sign. Hence \(F\) is strictly monotone, so there is exactly one root for every \(b\).

2. **\(|a|=1\)**:  
   Still \(F'\) has one sign (zero only at one point), so \(F\) is strictly monotone. Again, exactly one root for every \(b\).

3. **\(a=0\)**:  
   \(\log(1+x^2)=b\) has exactly one real solution only when \(b=0\) (then \(x=0\)).

4. **\(0<|a|<1\)**:  
   Write \(s=\sqrt{1-a^2}\). Then \(F'=0\) at two points, producing one local min and one local max, so there is exactly one root iff both extrema are on the same side of \(0\). The threshold values are
   \[
   m(a)=\log\!\left(\frac{2}{1+s}\right)-1+s,\qquad
   M(a)=\log\!\left(\frac{2}{1-s}\right)-1-s.
   \]
   Hence there is exactly one root iff
   \[
   b<m(a)\quad\text{or}\quad b>M(a).
   \]

So the set is
\[
\{(a,b): |a|\ge 1\}\;\cup\;\{(0,0)\}\;\cup\;
\left\{(a,b):0<|a|<1,\ b<m(a)\ \text{or}\ b>M(a)\right\}.
\]
