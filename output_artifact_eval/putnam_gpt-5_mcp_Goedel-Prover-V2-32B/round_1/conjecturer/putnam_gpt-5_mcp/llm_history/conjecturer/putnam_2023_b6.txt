where \(N(a,b)\) counts \(r\in\{1,\dots,n-1\}\) such that \(ar\le n\) and \(b(n-r)\le n\).
A short case check shows:
- \(N(1,1)=n-1\),
- \(N(1,b)=\lfloor n/b\rfloor\) for \(b>1\),
- \(N(a,1)=\lfloor n/a\rfloor\) for \(a>1\),
- \(N(a,b)=1\) only for \(a=b=2\) when \(n\) is even, and otherwise \(0\).

Thus
\[
w^TRw=(n-1)+2\sum_{k\ge2}\mu(k)\Bigl\lfloor\frac nk\Bigr\rfloor+\mathbf 1_{2\mid n}.
\]
Using the standard identity
\[
\sum_{k=1}^n \mu(k)\Bigl\lfloor\frac nk\Bigr\rfloor=1,
\]
we get
\[
w^TRw=1-n+\mathbf 1_{2\mid n}.
\]
Therefore
\[
2-w^TRw=
\begin{cases}
n,& n\text{ even},\\
n+1,& n\text{ odd},
\end{cases}
= n+(n\bmod 2).
\]

Finally,
\[
\det(R)=(-1)^{(n-1)(n-2)/2},
\]
so
\[
\det S = (-1)^{(n-1)(n-2)/2}\,\bigl(n+(n\bmod 2)\bigr).
\]

This matches the computed values
\[
2,2,-4,-4,6,6,-8,-8,\dots
\]

So a valid Lean answer is:

<<<fun n : ℕ => ((-1 : ℤ) ^ (((n - 1) * (n - 2)) / 2)) * (((n + n % 2 : ℕ) : ℤ))>>>
