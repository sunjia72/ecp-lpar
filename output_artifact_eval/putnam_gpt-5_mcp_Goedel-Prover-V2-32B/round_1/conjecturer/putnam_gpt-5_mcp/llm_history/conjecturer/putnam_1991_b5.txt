Let  
\[
I_p=\{z\in \mathbb F_p: z=x^2 \text{ for some }x\}\cap \{z\in \mathbb F_p: z=y^2+1 \text{ for some }y\}.
\]
We want \(|I_p|\) for odd prime \(p\).

A point \(z\in I_p\) means
\[
z=x^2=y^2+1,
\]
so
\[
x^2-y^2=1 \quad\iff\quad (x-y)(x+y)=1.
\]
Thus if we set \(t=x+y\in \mathbb F_p^\times\), then \(x-y=t^{-1}\), so
\[
x=\frac{t+t^{-1}}2,\qquad y=\frac{t-t^{-1}}2.
\]
Hence every such \(z\) has the form
\[
z=x^2=\left(\frac{t+t^{-1}}2\right)^2.
\]

Now put \(w=t^2\), so \(w\) runs over the nonzero quadratic residues \(Q\subset \mathbb F_p^\times\). Then
\[
z=\frac{(w+1)^2}{4w}=:g(w).
\]
Also
\[
g(w)=g(w^{-1}).
\]
In fact, if \(g(a)=g(b)\), then
\[
\frac{(a+1)^2}{a}=\frac{(b+1)^2}{b}
\]
and simplifying gives
\[
(a-b)(ab-1)=0,
\]
so the only identifications are \(a=b\) or \(a=b^{-1}\).

Therefore \(|I_p|\) is the number of orbits of \(Q\) under \(w\mapsto w^{-1}\).

Now \(|Q|=\frac{p-1}{2}\). Fixed points satisfy \(w=w^{-1}\), so \(w^2=1\), hence \(w=\pm1\).  
- \(w=1\) is always a square.  
- \(w=-1\) is a square iff \(p\equiv1\pmod 4\).

So the number of fixed points is:
- \(1\) if \(p\equiv3\pmod4\),
- \(2\) if \(p\equiv1\pmod4\).

Thus the number of orbits is
\[
\frac{|Q|+\#\text{fixed}}2.
\]
Hence
- if \(p\equiv3\pmod4\),
  \[
  |I_p|=\frac{(p-1)/2+1}{2}=\frac{p+1}{4};
  \]
- if \(p\equiv1\pmod4\),
  \[
  |I_p|=\frac{(p-1)/2+2}{2}=\frac{p+3}{4}.
  \]
