The supercontinuous functions are exactly those that are **affine on** \([0,1]\); outside \([0,1]\) they are unrestricted.

Python check idea: for a non-affine example \(f(x)=x^2\), alternate huge blocks of all \(1/2\) with huge blocks containing equal numbers of \(0\) and \(1\).  
Each block has average \(1/2\), so the original sequence is Cesàro-convergent to \(1/2\). But the \(f\)-block averages are \(1/4\) and \(1/2\), so the transformed sequence is not Cesàro-convergent. The printed experiment above matches this.

Informal proof:
- If \(f(x)=ax+b\) on \([0,1]\), then
  \[
  \frac1n\sum_{i<n} f(x_i)=a\frac1n\sum_{i<n}x_i+b,
  \]
  so Cesàro convergence is preserved.
- Conversely, assume \(f\) is supercontinuous.  
  For any \(u,v\in[0,1]\) and rational \(\lambda\in[0,1]\), let \(y=\lambda u+(1-\lambda)v\).  
  If \(f(y)\neq \lambda f(u)+(1-\lambda)f(v)\), alternate very long blocks of:
  1. all entries equal to \(y\),
  2. entries consisting of the correct rational proportions of \(u\) and \(v\),
  
  so every block has average \(y\). Then the original sequence has Cesàro limit \(y\), but the transformed block averages alternate between two different values, contradiction. Hence
  \[
  f(\lambda u+(1-\lambda)v)=\lambda f(u)+(1-\lambda)f(v)
  \]
  for all rational \(\lambda\).
- Taking \(u=1,v=0\), we get \(f(q)=aq+b\) for every rational \(q\in[0,1]\), where \(a=f(1)-f(0)\), \(b=f(0)\).
- For irrational \(x\in[0,1]\), choose rationals \(q_n\to x\). Alternate huge blocks of all \(x\)'s with huge blocks of all \(q_n\)'s. The original sequence still has Cesàro limit \(x\). If \(f(x)\neq ax+b\), then the transformed Cesàro averages oscillate between \(f(x)\) and \(aq_n+b\to ax+b\), contradiction. So \(f(x)=ax+b\) too.

Thus the solution set is exactly all functions affine on \([0,1]\).
