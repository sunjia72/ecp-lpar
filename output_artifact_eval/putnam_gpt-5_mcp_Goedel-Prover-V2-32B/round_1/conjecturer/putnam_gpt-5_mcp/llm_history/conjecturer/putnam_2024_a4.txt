Let  
\[
S=\{a^{e(n)} \bmod p : 0\le n\le p-5\}.
\]
Since
\[
a^{e(n)}-a^{e(n-1)}\equiv r \pmod p
\]
for all \(1\le n\le p-5\), the values in \(S\) form an arithmetic progression of length \(p-4\) in \(\mathbb F_p\), with common difference \(r\neq 0\). Because \(e\) is a bijection on \(\{0,\dots,p-5\}\), this means
\[
S=\{a^0,a^1,\dots,a^{p-5}\}.
\]
These are distinct, so \(a\) has order \(p-1\), i.e. \(a\) is a primitive root mod \(p\).

Thus the complement of \(S\) in \(\mathbb F_p\) is
\[
\{0,a^{-1},a^{-2},a^{-3}\},
\]
and since the complement of a long arithmetic progression is also a 4-term arithmetic progression, \(\{0,a^{-1},a^{-2},a^{-3}\}\) is a 4-term AP. Multiplying by \(a^3\), so is
\[
\{0,1,a,a^2\}.
\]

Now classify 4-term APs containing \(0\). Checking the possible positions of \(0\) in the progression:

- If \(0\) is an interior term, then \(\{1,a,a^2\}\) must be one of sets like \(\{-d,d,2d\}\) or \(\{-2d,-d,d\}\). A short case check gives only \(p=3\) or \(p=5\), impossible since \(p>5\).
- So \(0\) must be an endpoint, and \(\{1,a,a^2\}=\{d,2d,3d\}\). Since \(1\) is one of these, \(d\in\{1,\tfrac12,\tfrac13\}\). Checking the resulting possibilities, the only prime \(>5\) that works is \(p=7\), with \(a=3\) or \(a=5\).

Indeed for \(p=7\), take \(a=3\), \(r=1\), and define \(e\) on \(\{0,1,2\}\) by
\[
e(0)=0,\quad e(1)=2,\quad e(2)=1.
\]
Then
\[
3^{e(0)}=1,\quad 3^{e(1)}=2,\quad 3^{e(2)}=3 \pmod 7,
\]
so consecutive differences are all \(1\).

So the solution set is exactly \(\{7\}\).
