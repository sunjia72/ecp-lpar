Let \(a_m = p(m)\).  
The condition says that for each \(m \ge 2\), the new value \(a_m\) differs by \(1\) from some earlier value \(a_k\) with \(k<m\).

So after the first \(m\) terms, the set
\[
S_m=\{a_1,\dots,a_m\}
\]
must always be a connected subset of the path graph \(1-2-\cdots-n\). Hence \(S_m\) must be an interval of consecutive integers.

Starting from \(a_1=s\), the set grows from \(\{s\}\) to \([1,n]\) by repeatedly adding either the next unused value on the left or the next unused value on the right.  
There are:

- \(s-1\) left-expansions,
- \(n-s\) right-expansions,

for a total of \(n-1\) steps. Thus for fixed starting value \(s\), the number of valid permutations is
\[
\binom{n-1}{s-1}.
\]

Summing over all possible starts \(s=1,\dots,n\),
\[
\sum_{s=1}^n \binom{n-1}{s-1} = 2^{n-1}.
\]

So the solution function is \(n \mapsto 2^{n-1}\) (and at \(n=0\), Lean’s natural subtraction makes this \(2^0=1\), which is fine since the theorem only requires \(n>0\)).
